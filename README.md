# hantenabookmark-replace-tags

A CLI that replace all hatena bookmark tags.

## Install

Install with [npm](https://www.npmjs.com/):

    npm install hantenabookmark-replace-tags

## Usage


1. Get OAuth Token
    - `yarn run get-token`
2. Run scripts
    - `yarn run .....`
